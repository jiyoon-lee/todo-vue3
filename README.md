# Todos
vue3를 활용하여 Todo 애플리케이션을 만들어봄을써 학습하였습니다.

### 주요 기능
- todo 조회, 추가, 삭제, 수정
- todo 활성화/비활성화 변경
- 활성화된 todos만 조회
- 완료된 todos만 조회
- 전체 지우기

### 사용기술
vue3, bootstrap, vite, prettier, eslint
